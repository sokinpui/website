---
title: How to setup terminal environment
desc: Documentation for setup terminal config for new *nix device
---

run this script:

```bash
wget -O- https://raw.githubusercontent.com/sokinpui/terminal_dotfiles/refs/heads/main/zsh/setup | bash
```

It should install:

- zsh
- tmux
- neovim
- fd
- ripgrep
- fzf
- git
- curl
- pipx
- zoxide
- pyenv
- direnv
- lf
- wtgo
- itf
- pcat
- trash-cli
